# Multiplayer Game Server (Node.js + UDP)

Этот сервер реализует логику для небольшой многопользовательской игры на Unity.

## Особенности
- **UDP Протокол**: Минимальная задержка.
- **Серверная авторитарность**: Движение, стрельба и расчет попаданий происходят на сервере.
- **Система HP**: У каждого игрока 3 жизни.
- **Рестарт**: Возможность возрождения после проигрыша.

## Как запустить
1. Установите Node.js (если еще не установлен).
2. Откройте терминал в папке `Assets/Scripts/`.
3. Запустите сервер командой:
   ```bash
   node Server.js
   ```

## Протокол взаимодействия (JSON через UDP)

### Клиент -> Сервер (Порт 7777)
Отправляется каждый кадр или при событии:
```json
{
  "id": "Player_1",
  "inputX": 1.0,
  "inputY": 0.0,
  "isShooting": true,
  "dirX": 1.0,
  "dirY": 0.5,
  "restart": false
}
```

### Сервер -> Клиент
Рассылается всем игрокам 30 раз в секунду:
```json
{
  "players": {
    "Player_1": { "id": "Player_1", "x": 5.2, "y": 1.0, "hp": 3 },
    "Player_2": { "id": "Player_2", "x": -2.1, "y": 0.5, "hp": 1 }
  },
  "bullets": [
    { "x": 1.5, "y": 2.0 },
    { "x": 3.2, "y": 1.1 }
  ]
}
```
